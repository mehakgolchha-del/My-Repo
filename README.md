# 🤖 AI Assistant - Multi-Feature Personal AI

A comprehensive AI assistant with voice control, PC automation, coding assistance, image/video processing, and business analytics.

## Features

### 🎙️ Voice Assistant with Offline LLM
- Wake word detection ("Hey Vish")
- Text-to-speech responses with male voice
- Offline LLaMA 2 / Mistral integration
- Voice command understanding and execution

### 🖥️ PC Automation & Productivity
- App control (open/close applications)
- File management operations
- Screenshot capture
- System information retrieval
- Task automation

### 💻 AI-Powered Coding Tools
- Code generation and completion
- Debugging assistance
- Multi-language support (Python, Java, C++, JavaScript, etc.)
- Code review and optimization

### 🎨 Image & Video Processing
- Image enhancement and filters
- Background removal
- Face detection and retouching
- Video trimming and effects
- AI-powered subtitle generation

### 📊 Business Analytics Dashboard
- Income and expense tracking
- Financial reporting
- Business metrics visualization
- Trend analysis
- Performance insights

## Project Structure

```
├── voice_assistant/       # Voice control and LLM integration
├── pc_automation/         # PC automation tools
├── coding_assistant/      # AI coding tools
├── media_processing/      # Image and video processing
├── analytics_dashboard/   # Business analytics

```
